## Key Features
- **Strict Scheduling**: Automated 15-minute slot management.
- **Conflict Prevention**: Prevents overlapping or duplicate slots.
- **Auto-Cleanup**: Automatically hides past slots from the dashboard.
- **Real-time Notifications**: Premium SweetAlert2 integration for all actions.
- **Social Auth**: Google login integration.
- **Modern UI**: Clean, responsive design with glassmorphism aesthetics.

## Tech Stack
- **Frontend**: Next.js 15, Tailwind CSS, SweetAlert2.
- **Backend**: Node.js, Express.
- **Database**: MongoDB (Native Driver).
- **Auth**: Firebase & JWT.
